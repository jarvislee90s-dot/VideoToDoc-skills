## 📌 短摘要

Anthropic 承认对 Claude 模型静默降智错误并转为可见机制；OpenAI 大幅降价被解读为缺乏优势的被动防守；MiniMax 宣布开源 M3 模型权重；DeepSeek 开放 Agent Harness；Meta 与 Manus 完成运营分离。

---

## 📝 详细摘要

### 1️⃣ Anthropic 认错：撤回 Claude 静默降智策略

- Anthropic **撤回了对 Claude Fable 5 模型的静默破坏策略**（即对大模型开发相关请求进行暗中介入降智）
- **承认此前不透明的做法是错误的权衡**
- 从本周起，相关防护机制**转为可见**：
  - 系统会**明确拒绝**相关请求
  - 或显示**回退至较弱的 OP4.8 模型**
  - API 端也会**返回具体原因**
- 官方坦言：转为可见机制后，短期内可能**增加无害请求的误报率**

### 2️⃣ OpenAI 大幅降价：被动防守还是主动出击？

- OpenAI **大幅降价**以争夺 Anthropic 等对手的用户
- 社区讨论指出：这或是**缺乏压倒性信心模型的被动防守**
- 其**烧钱抢客与盈利可持续性正遭到质疑**

### 3️⃣ MiniMax M3 即将开源

- MiniMax 官方宣布**开源高性能 MSA Kernel 库**
- 预计**本周五公开 MiniMax M3 模型权重**

### 4️⃣ Midjourney 升级至 V8.1

- Midjourney 官方宣布默认模型从 V7 更新为 **V8.1**
- 新模型提升**连贯性与文本渲染能力**
- HD 模式下分辨率达 **V7 的 4 倍**

### 5️⃣ 阿里云 Mio CLI 开源

- 阿里云旗下云端 AI 开发工具**秒悟 Mio**推出开源命令行工具 **Mio CLI**
- 支持将本地生成的代码**一键部署至云端**
- 直接接管数据库和文件存储等后端服务

### 6️⃣ DeepSeek 开放 Agent Harness

- DeepSeek 开放了 **Agent Harness**（研究员职位招聘同步进行）
- 旨在探索**上下文管理、Multi Agents** 等前沿领域
- 目标：将前沿模型能力转化为**领先的 Agent 产品**

### 7️⃣ Meta 与 Manus 完成运营分离

- 为落实中国监管部门撤销收购案的要求
- Meta 已与 Manus **完成运营分离**
- 切断双方数据共享及内部系统访问权限

### 8️⃣ 字节跳动人事变动

- 据 ZFinance 报道，字节跳动豆包手机硬件产品负责人**林希已离职**
